## Project setup 
```
npm install
```

### Run the app
```
npm start
```

### Install Electron packager
```
npm install electron-packager -g
```

### Compiles for production
```
npm run package-mac
```
```
npm run package-win
```
```
npm run package-linux
```

### Create Mac installer
```
npm install electron-installer-dmg -g
```
```
npm run create-installer-mac
```
